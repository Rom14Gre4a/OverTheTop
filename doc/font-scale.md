# Font Scale System

## Огляд

Усі розміри шрифтів у проєкті централізовані через CSS custom properties і керуються через сторінку **Налаштування → 🔡 Розмір тексту**.

---

## CSS-змінні

| Змінна          | Призначення                          | Large (default) |
|-----------------|--------------------------------------|-----------------|
| `--fz-display`  | Великі числа, екрани завершення      | 40 px           |
| `--fz-h1`       | Заголовки сторінок                   | 28 px           |
| `--fz-h2`       | Підзаголовки / назви карток          | 21 px           |
| `--fz-body`     | Основний текст                       | 16 px           |
| `--fz-sm`       | Допоміжний / описовий текст          | 14 px           |
| `--fz-xs`       | Мітки, бейджі, комірки таблиць       | 12 px           |
| `--fz-micro`    | Дрібні підписи, таймстемпи, мета     | 11 px           |

Змінні визначені в `frontend/app/globals.css` (`:root`) і перевизначаються через `document.documentElement.style.setProperty(...)` у `ThemeContext`.

---

## Використання у коді

### Inline-стилі (JSX)

```tsx
// ✅ Правильно — масштабується з налаштувань
<span style={{ fontSize: "var(--fz-sm)" }}>Текст</span>

// ❌ Неправильно — не масштабується
<span style={{ fontSize: 13 }}>Текст</span>
```

### Tailwind-класи

Tailwind `text-*` класи автоматично прив'язані до CSS-змінних через `@theme inline` у `globals.css`:

| Tailwind клас  | CSS-змінна      |
|----------------|-----------------|
| `text-xs`      | `--fz-micro`    |
| `text-sm`      | `--fz-xs`       |
| `text-base`    | `--fz-body`     |
| `text-lg`      | `--fz-sm`       |
| `text-xl`      | `--fz-h2`       |
| `text-2xl`     | `--fz-h1`       |
| `text-3xl`     | `--fz-display`  |

## Пресети

Керуються через `ThemeContext` → `setFontPreset(preset)`.  
Зберігаються в `localStorage` (`ott-font-preset`, `ott-font-custom`).

| Пресет   | display | h1 | h2 | body | sm | xs | micro |
|----------|---------|----|----|----- |----|----|----|
| compact  | 32      | 22 | 16 | 13   | 11 | 10 |  9 |
| normal   | 36      | 26 | 19 | 15   | 13 | 11 | 10 |
| **large** *(default)* | **40** | **28** | **21** | **16** | **14** | **12** | **11** |
| xl       | 48      | 34 | 25 | 19   | 16 | 14 | 12 |

Після вибору пресету можна точно налаштувати кожен тип окремо через **Детальне налаштування** (слайдери, діапазон 7–60 px).

## Де застосовано

Скрипт `scripts/replace-font-sizes.ps1` (запускався одноразово при введенні системи) замінив усі hardcoded `fontSize: N` у `.tsx`/`.ts` файлах на `fontSize: "var(--fz-X)"` за таким маппінгом:

| Діапазон px | CSS-змінна      |
|-------------|-----------------|
| 7–9         | `--fz-micro`    |
| 10–11       | `--fz-xs`       |
| 12–13       | `--fz-sm`       |
| 14–16       | `--fz-body`     |
| 17–22       | `--fz-h2`       |
| 23–30       | `--fz-h1`       |
| 31+         | `--fz-display`  |

Зачеплені файли: `page.tsx` (тренування, сесія, деталі, нова сесія, дашборд, граф, альманах, логін, реєстрація, адмін), `Sidebar.tsx`, `ThemeSwitcher.tsx`, `Modal.tsx`, `TierListBoard.tsx`.

## Додавання нових компонентів

При написанні нових компонентів **завжди використовуй CSS-змінні**, не числа:

```tsx
// Орієнтир вибору змінної:
// великий заголовок сторінки → var(--fz-h1)
// назва картки / секції      → var(--fz-h2)
// основний текст             → var(--fz-body)
// підказки / описи           → var(--fz-sm)
// бейджі / мітки / теги      → var(--fz-xs)
// таймстемп / мета / помітки → var(--fz-micro)
// числа на екрані завершення → var(--fz-display)
```
