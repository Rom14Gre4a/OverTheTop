# Roadmap — Дорожня карта

> Пріоритети: 🔴 Критично | 🟠 Важливо | 🟡 Середньо | 🔵 UX | ⚪ Довгострокове

---

## ФАЗА 1 — Core Loop (Журнал тренувань)
> Без цього аналітика і графіки безглузді. Стартова точка.

### Бекенд
- [ ] 🔴 `ExerciseSet` entity — нова таблиця підходів (reps, weightKg, isPersonalRecord)
- [ ] 🔴 `TrainingSessionController` — CRUD `/api/sessions`
- [ ] 🔴 Автодетект PR при збереженні підходу (порівняння з попереднім max weightKg)
- [ ] 🔴 EF міграція для `exercise_sets`, `weight_logs`, `recovery_logs`

### Фронтенд
- [ ] 🔴 Сторінка `/today` — "Сьогодні": план на день + швидкий старт
- [ ] 🔴 Сторінка `/training/log` — список всіх сесій
- [ ] 🔴 Сторінка `/training/log/new` — логування сесії (вправи → підходи → вага×reps)
- [ ] 🔴 Таймер відпочинку між підходами
- [ ] 🔴 `lib/sessions.ts` — API клієнт для сесій
- [ ] 🔴 Мітка "PR" при збереженні рекордного підходу
